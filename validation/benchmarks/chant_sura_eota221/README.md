# Chant Sura EOTA221 Shape Benchmark

Role: passive shape/orientation metadata readiness for future shape-contact
validation. This benchmark must not be interpreted as active shape-aware
physics while the simulator contact model remains spherical.

Preparation scaffold:

```bash
python3 scripts/prepare_chant_sura_eota221_benchmark.py
```

The scaffold summarizes public EOTA shape metadata already processed into the
small checked-in `rock_shapes.csv` fixture when available.
