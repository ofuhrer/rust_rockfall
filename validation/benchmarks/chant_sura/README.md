# Chant Sura Public Benchmark

Role: trajectory/contact validation using public reconstructed trajectories,
small DEM-backed contact fixtures, impact-timing proxies, jump-height envelopes,
and energy-evolution diagnostics.

Preparation scaffold:

```bash
python3 scripts/prepare_chant_sura_public_benchmark.py
```

The scaffold records the checked-in small processed fixtures and existing
validation cases. It does not download the large public EnviDat archives.
